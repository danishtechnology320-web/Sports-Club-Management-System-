# 🏏 Sports Club Management System

A web-based management system designed to streamline operations of a sports club.  
This project helps manage members, events, payments, and club activities with a clean 
interface and organized architecture.

---

## 📌 Features
- **Member Management** – Add, update, and track club members.
- **Event Scheduling** – Create and manage sports events with details.
- **Payment Tracking** – Record membership fees and other payments.
- **Responsive UI** – Styled with `entypo.css` icons and modern design.
- **Role-Based Access** – Admin and user-level functionalities.

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS (Entypo icons), JavaScript  
- **Backend:** Java (Spring Boot)  
- **Database:** MySQL with JPA/Hibernate  
- **Templating:** Thymeleaf / JSP
